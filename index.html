TrayX is a lightweight Windows application that adds informative icons to the tray

Features

Workflow-friendly

Runs quietly in the background, always ready when you need system stats, without interrupting your tasks.

Real-time Graphs

Visualize data with dynamic graphs, making it easier to monitor system performance and track changes over time.

Information Panels

Compact pop-up panels that appear when clicking the icon, providing essential data (e.g., CPU/GPU temperature, memory and disk usage).

Getting started

  1. Download TrayX
    • Click the “Download” button on our website to get the ZIP archive.
  2. Extract the Archive
    • Unpack the contents into any folder you prefer. No installation required — TrayX runs without setup.
  3. Launch the Application
    • Run TrayX.exe. After starting, its icon will appear in the system tray.
  4. Customize Your Experience
    • Open the main menu (Right-click the tray icon → “Open Menu”) to:
    • Change the interface language;
    • Enable or disable specific features;
    • Adjust how tray icons are displayed.
  5. Runs in the Background
    • Closing the main window does not quit the app; it keeps running in the tray.
    • To fully exit TrayX, right-click the icon and select “Exit.”
